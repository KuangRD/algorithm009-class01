## 布隆过滤器

一个长二进制向量，一系列随机散列函数。
查找是否可能存在（可能存在/不可能存在）


Pros ： 空间效率高，查询时间快

Cons ： 查全率高，查准率一般，需要进一步确认， 删除困难。
